一.java文件介绍：
Demo.java：主类，通过运行这个类来生成浏览器，里面封装了主界面按钮以及按钮的事件，还有工具函数等
BookMark.java：书签的类
downloadfile.java下载功能（断点续传）的功能类
E_mail.java：发邮件的类
fileUI.java：下载界面的界面类
History.java：历史记录的类
Log.java：日志类
savefile.java：保存页面的适配器类，类似于中介

二.运行方式：
1.将Browser1文件夹导入idea：
打开idea编译器，选择file->new->project form existing source；然后选择Browser1文件夹导入即可，后面如果还有要选择的话就选择project form existing source。
2.导入后运行Demo.java文件，就会启动浏览器
